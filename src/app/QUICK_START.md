# 🚀 Tod AI - Quick Start Guide

## Welcome to Tod AI! 

Your adaptive AI learning companion powered by Google Gemini is ready to help you learn anything, anywhere!

## 📋 Quick Setup (5 Minutes)

### Step 1: Login
- Use demo credentials: `student@todai.com` / `demo123`
- Or create a new account

### Step 2: Choose Your Type
- **School Student (K-8)**: Fun, colorful interface with games
- **College Student**: Professional interface with advanced tools

### Step 3: Setup AI (First Time Only)
1. Click the ✨ sparkle button (bottom-right)
2. Get your free API key: [Google AI Studio](https://aistudio.google.com/app/apikey)
3. Enable the API: [Enable Gemini API](https://console.cloud.google.com/apis/library/generativelanguage.googleapis.com)
4. Paste your key and click "Test & Save"
5. Done! 🎉

## 🆘 Troubleshooting

### AI Not Responding?
- Check API key is saved (click sparkle → settings)
- Verify internet connection
- Make sure Gemini API is enabled
- Try refreshing the page

### Voice Not Working?
- Grant microphone permission
- Use Chrome or Edge browser
- Check microphone is connected
- Click mic icon and speak clearly

### API Key Error?
- Verify key is correct (starts with "AIza")
- Enable Generative Language API
- Wait 1-2 minutes after enabling
- Get a new key if needed
